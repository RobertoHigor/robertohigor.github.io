Trabalho de conclusão de curso

Consistiu em um projeto de fechadura eletrônica e controle de acesso utilizando Arduino. Para capturar as senhas, foi utilizado um módulo de teclado. O projeto foi utilizado no laboratório da FAETERJ - Petrópolis.

Cada usuário no sistema possui uma senha customizada, que é gerada após a realização do cadastro. A página de cadastro também é responsável por exibir os acessos, caso o usuário tenha a permissão de visualizar, a fim de detectar acessos indevidos as salas.

O projeto ao todo possui em 3 aplicações, sendo então:
  • Aplicação Desktop em Python para comunicação com o banco de dados;
  • Aplicação web com Django para cadastro e listagem de acessos;
  • Aplicação Arduino para envio dos dados de um teclado numérico;
